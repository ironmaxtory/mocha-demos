## How to use
First, clone the repo.
```bash
$ git clone https://github.com/ironmaxtory/mocha-demos.git
```

Then, install the dependencies locally and Mocha globally.
```bash
$ cd mocha-demos
$ npm install
$ npm install -g mocha
```

After, run the scripts that has been listed in the `package.json`.
Like this:
```bash
$ npm run test
```
